# Code of Conduct

This project adopts a simple standard: be respectful, constructive, and professional.

## Expected Behavior

- Be respectful in discussions and reviews.
- Focus on technical issues, not people.
- Give clear, actionable feedback.

## Unacceptable Behavior

- Harassment, personal attacks, or discrimination
- Trolling or intentionally disruptive behavior
- Sharing private or confidential data without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or block users who violate this policy.
